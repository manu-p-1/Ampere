    CheckContains()

    Checks whether each array specified in otherArrays is completely contained in the base array. Each value in each array must be contained in the base array for this method to return true, otherwise, false is returned and the ViolatedEnumerable property is set with the enumerable that violated the condition. This function runs a LINQ intersection between each enumerable.
