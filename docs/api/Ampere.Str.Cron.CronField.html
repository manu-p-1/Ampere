Class CronField

Namespace
Ampere.Str.Cron
Assembly
Ampere.dll

Represents a single parsed field (e.g. minutes, hours) of a cron expression. A field contains one or more CronNode elements (the comma-separated parts). Instances are immutable and thread-safe.
