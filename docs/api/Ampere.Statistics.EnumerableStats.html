Class EnumerableStats

Namespace
Ampere.Statistics
Assembly
Ampere.dll

A static utility class for calculating a variety of statistics based on IEnumerables. Moreover, generic overload methods exist to simplify calculating statistics of object enumerables containing numeric properties.
