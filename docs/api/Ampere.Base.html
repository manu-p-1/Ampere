IntRange

The range class represents a range of int values. Unlike other IRangify<T> implementing classes, IntRange contains an System.Collections.Generic.IEnumerator<T> of type int to enumerate through all of the values between the minimum and maximum ranges. By convention, both sides of the range should be inclusive values.
